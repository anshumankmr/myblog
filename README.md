# JAMStack (Next.js + Google Cloud)

This project has been rewritten to use **Next.js** for the frontend while still
serving content from the same Strapi backend hosted on Google Cloud.

## Getting Started

1. `git clone https://github.com/anshumankmr/myblog`
2. `cd myblog/next-blog`
3. `npm install`
4. `npm run dev`

Visit it here at [anshumankumar.dev](https://www.anshumankumar.dev/).

I have also written a blog about the original solution architecture. The link is [here](https://www.anshumankumar.dev/article/2023-08-31/building-a-blog-with-cloud-and-coding?articleId=b005d32b-5eef-4804-a0ca-13f21c742daa). Please do check it out.
